# Problem Statement
Tracking daily expenses becomes difficult when done manually. People often forget how much money they have spent in different categories, and it becomes confusing to remember total spending. There is a need for a simple tool where expenses can be added and reviewed quickly without complexity.

# Scope of the Project
This project provides a simple Python-based tool to record daily expenses with amount, category, note and date. The data is safely stored in a JSON file and can be viewed, deleted and summarized anytime.

# Target Users
Anyone who wants to manage personal spending easily. This project is especially useful for students, individuals living independently, or anyone trying to improve budgeting and financial awareness.

# High-Level Features
• Add a new expense with amount, category, note and date  
• View all saved expenses in list format  
• Delete an expense by ID  
• View category-wise summary of spending  
• Check total money spent
