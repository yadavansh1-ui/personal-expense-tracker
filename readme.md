# Personal Expense Tracker

This is a small Python program I made for tracking daily expenses.  
It runs in the terminal and saves the data in a simple JSON file.

## Technologies Used
- Python
- JSON File Storage

## Features
- Add a new expense (amount, category, note, date)
- View all saved expenses
- Delete an expense using its ID
- Category-wise summary of spending
- Total money spent
- Supports multiple date formats
- Data stored safely in a JSON file

## How to Run
1. Install Python 3 on your system  
2. Download all project files  
3. Open the terminal inside the project folder  
4. Run the program using:

```python
python expense.py
```

## Files Included
- `expense.py` — main program file  
- `expenses.json` — stores the expense data  
- `README.md` — documentation file  
- `statement.md` — project explanation file

